# simple-intrusion-detection-system
A simple intrusion detection system that detects multiple failed login attempts
A beginner cybersecurity project built using python.

## What this project does 
This program analyzes a security log file and detects suspicious activity based on failed login attempts.

## Features 
- Read a log file
- Counts failed login attempts
- Generates an alert when multiple failed logins are detected
- Demonstrated basic intrusion detection concepts

## Technologies Used
- Python 3

## Files 
- simple_ids.py
- security_log.txt

## How to Run
python simple_ids.py
